# Cyber_Bot

A modern **Cyber Security Awareness Chatbot** built using **C# WPF** in Visual Studio.

The chatbot helps users learn basic cybersecurity concepts such as:

* Password Safety
* Phishing Awareness
* Online Privacy
* Scam Protection

The application includes:

* GUI Interface (WPF)
* Dynamic Responses
* Sentiment Detection
* Memory Features
* Sound Greeting
* Custom CyberBot Logo

---

# Features

## GUI Interface

* Built with WPF
* Dark cyber-themed design
* Interactive chat interface

## Cybersecurity Topics

The bot responds to:

* Passwords
* Phishing
* Privacy
* Scams

## Sentiment Detection

The bot can detect emotions like:

* Worried
* Curious
* Frustrated

## Memory

The bot remembers:

* User name
* Favourite topic

## Multimedia

* Custom logo image
* Greeting sound on startup

# How to Run the Project

## Step 1

Open the project in:

```text
Visual Studio
```

## Step 2

Make sure these files are added to the project:

* `logo.jpeg`
* `greeting.wav`

---

## Step 3

Set file properties:

### logo.jpeg

```text
Build Action = Content
Copy to Output Directory = Copy if newer
```

### greeting.wav

```text
Build Action = Content
Copy to Output Directory = Copy if newer
```

---

## Step 4

Run the project:

```text
Press F5
```
